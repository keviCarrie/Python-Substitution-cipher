#Python substitution cipher

A simple, interactive Python tool that uses a Substitution Cipher to encypt and decrypt messages. 
This program generates a randomized key to scamble characters, making it a great examcharacter at the sample of basic cryptography.
features
Randomized encryption: generates a unique key by shuffling a character set(letters, digits, whitespace and punctuation)
Full character support: handles spaces, numbers and symbols for robust messaging.
How it works
The script creates a master list of all possible characters and then creates a sguffled version of the list(key).Each character in your original message is replaced by the character at the same position(index) in the shuffled list.
